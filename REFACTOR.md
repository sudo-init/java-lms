

### 요구사항
질문 데이터 삭제


### 기능
* [ ] 질문 삭제 (완전 삭제 X)  
* [ ] 답변 삭제 (완전 삭제 X)
* [ ] 데이터의 상태를 삭제 상태로 변경
* [ ] 삭제 이력 정보를 DeleteHistory에 남김.

* **삭제 조건**    
* [ ] 로그인 사용자와 질문한 사람이 같은지 확인 (항상)
* [ ] 답변이 없는지 확인 (Case1)
* [ ] 질문자와 답변 글의 모든 답변자가 같은 경우 (Case 2)


### 제약
* [ ] 질문자와 답변자가 다른 경우 답변을 삭제할 수 없음.
